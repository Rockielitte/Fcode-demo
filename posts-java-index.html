MỞ ĐẦU:

Sau khi tìm hiểu xong phần 1 thì các bạn đã biết cách tạo 1 web service sử dụng RESTful API và Java Persistence API (JPA). Ở phần này mình sẽ giới thiệu các bạn sử dụng XMLHttpRequest (XHR) để lấy dữ liệu từ web service và sử dụng Javascript để xử lí dữ liệu.
